### 1. Agents Service测试 ✅

**问题**: CacheService依赖缺失

**修复**:
```typescript
// 添加CacheService mock
const mockCacheService = {
  get: jest.fn(),
  set: jest.fn(),
  del: jest.fn(),
  invalidate: jest.fn(),
  getOrSet: jest.fn((_key, factory, _ttl) => factory()),
};

// 在providers中添加
{
  provide: CacheService,
  useValue: mockCacheService,
}
```
