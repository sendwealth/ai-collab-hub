# 用户流程测试验证清单

## 文档说明
本文档用于验证AI协作平台的用户体验流程是否符合设计规范。

**版本**: v1.0  
**最后更新**: 2026-03-15  
**测试环境**: http://localhost:3001

---

## 🧪 测试环境准备

### 后端服务
- [ ] 启动后端 API 服务（端口 3000）
  ```bash
  cd apps/server
  pnpm dev
  ```

### 前端服务
- [ ] 启动前端应用（端口 3001）
  ```bash
  cd apps/web
  pnpm dev
  ```

### 数据库
- [ ] 确保数据库连接正常
- [ ] 运行数据库迁移（如有）
- [ ] 准备测试数据
