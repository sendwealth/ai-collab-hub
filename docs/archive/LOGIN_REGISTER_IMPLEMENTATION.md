## 🔧 技术实现

### 表单验证
```typescript
// 邮箱验证
/^[^\s@]+@[^\s@]+\.[^\s@]+$/

// 用户名验证
/^[a-zA-Z0-9_\u4e00-\u9fa5]+$/

// 密码强度检测（5项）
- 长度 ≥ 8
- 小写字母
- 大写字母
- 数字
- 特殊字符
```

### API集成
- 登录接口: `POST /api/v1/auth/login`
- 注册接口: `POST /api/v1/auth/register`
- Token存储:
  - 记住我: localStorage
  - 不记住: sessionStorage

### 状态管理
- React useState 管理表单状态
- 错误状态实时更新
- 加载状态控制
